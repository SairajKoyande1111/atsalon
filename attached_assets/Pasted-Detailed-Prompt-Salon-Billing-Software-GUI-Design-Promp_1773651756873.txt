Detailed Prompt: Salon Billing Software GUI Design

Prompt

Design a modern, professional, and intuitive Salon Billing Software GUI for desktop and tablet usage. The interface should be similar to a Point-of-Sale (POS) system used in salons, spas, and beauty clinics.

The UI should be clean, minimal, fast to operate, and optimized for touch screens.

Use a premium salon theme with colors like:

Primary: Deep Purple / Royal Gold

Secondary: Soft Beige / White

Accent: Rose Gold or Elegant Pink

Fonts: Poppins / Inter / Montserrat

The system must look like premium salon management software used in luxury salons.

Main Layout Structure

The application should have three main sections:

1. Left Sidebar (Navigation Menu)

A vertical navigation panel with icons and labels.

Menu Items:

Dashboard
New Bill / POS
Appointments
Customers
Services
Products
Staff
Inventory
Memberships
Offers / Coupons
Reports
Expenses
Settings
Logout

Each menu item should have a modern icon.

Sidebar should be collapsible.

2. Top Header Bar

The top bar should contain:

Salon Logo
Current Date & Time
Search Bar (Search Customer / Phone Number)
Notification Bell
Staff Profile / Login Name
Quick Add Button (+)

Example quick add options:

Add Customer
Add Appointment
Add Product
Add Service

3. Main Workspace Area

The main area changes depending on the selected module.

POS BILLING SCREEN (MOST IMPORTANT)

This is the core screen of the software.

Design a 3-column POS layout.

LEFT PANEL — Services & Products

This section shows:

Search Bar
Category Tabs

Example Categories:

Hair Services
Facial
Spa
Makeup
Nail Care
Hair Products
Skin Products

Below categories show service cards with:

Service Name
Price
Duration

Example card:

Haircut
₹300
30 min

Cards should be large touch friendly buttons.

CENTER PANEL — Bill Cart

Shows all selected services.

Columns:

Service Name
Staff
Price
Qty
Discount
Total

Each row should allow:

Edit price
Assign staff
Remove item

Below cart show:

Subtotal
Discount

Final Amount

Show:

Total Amount
